\# Frontend сервиса заметок



Клиентская часть на React.



\## Зависимости



\- React 18

\- React Router DOM

\- Axios



\## Компоненты



\- Login - форма входа

\- Register - форма регистрации

\- TwoFA - форма 2FA кода

\- NotesList - список заметок

\- NoteEditor - редактор заметок



\## Маршруты



| URL | Компонент |

|-----|-----------|

| /login | Login |

| /register | Register |

| /2fa | TwoFA |

| /notes | NotesList |

| /notes/new | NoteEditor |

| /notes/:id | NoteEditor |



\## Запуск



npm install

npm start



Приложение: http://localhost:3000
