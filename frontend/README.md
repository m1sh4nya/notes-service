\# Сервис заметок с 2FA



Веб-приложение для создания и управления заметками с двухфакторной аутентификацией.



\## 📁 Структура проекта

notes-service/

├── backend/ # Backend на FastAPI

├── frontend/ # Frontend на React

└── docs/ # Документация





\## 🚀 Технологии



\- \*\*Backend\*\*: Python + FastAPI + SQLite + JWT

\- \*\*Frontend\*\*: React + Axios

\- \*\*Auth\*\*: 2FA (TOTP)



\## 📝 Лабораторные работы



\- Лаба 1: Обоснование темы

\- Лаба 2: Архитектура

\- Лаба 3: Прототип

\- Лаба 4: Авторизация + 2FA

\- Лаба 5: Функциональный модуль (поиск)

\- Лаба 6: Асинхронность

\- Лаба 7: Тестирование
