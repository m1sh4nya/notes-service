# \# Сервис заметок с двухфакторной аутентификацией

# 

# \## О проекте

# 

# Веб-приложение для создания и управления заметками. Пользователи могут регистрироваться, входить в систему, создавать заметки и искать по ним. Для безопасности используется двухфакторная аутентификация.

# 

# \## Функционал

# 

# \- Регистрация и вход

# \- Двухфакторная аутентификация

# \- Создание, редактирование и удаление заметок

# \- Поиск по заметкам

# \- Хранение данных в базе

# 

# \## Технологии

# 

# \- Backend: Python + FastAPI

# \- Frontend: React

# \- База данных: SQLite

# \- Аутентификация: JWT + 2FA

# 

# \## Структура проекта

# 

# \- backend/ - серверная часть

# \- frontend/ - клиентская часть

# \- docs/ - документация

# 

# \## Запуск

# 

# Backend:

# cd backend

# python -m venv venv

# venv\\Scripts\\activate

# pip install -r requirements.txt

# python run.py

# 

# Frontend:

# cd frontend

# npm install

# npm start
